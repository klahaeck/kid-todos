# StarrySteps Brand Style Guide

## 1. Brand Overview

**Brand Name:** StarrySteps  
**Website:** [starrysteps.com](https://www.starrysteps.com)  
**Product:** A routine-management app for children, focused on morning and evening flows with fun buttons, animations, and an encouraging experience for families.

### Brand Personality
StarrySteps should feel:
- Playful
- Gentle
- Trustworthy
- Simple
- Encouraging
- Elegant

### Brand Promise
StarrySteps helps families turn daily routines into calm, positive, engaging moments.

### Emotional Tone
The experience should communicate:
- Safety
- Warmth
- Delight
- Progress
- Calm structure

---

## 2. Visual Brand Direction

### Core Concept
The brand should blend:
- Morning + bedtime
- Movement + progress
- Children + polish
- Fun + simplicity

### Suggested Visual Motifs
Use these as recurring visual cues:
- Stars
- Stepping stones / footprints / path dots
- Crescent moon
- Rising sun
- Sparkles
- Rounded clouds
- Soft motion arcs

### Design Style
- Rounded forms
- Soft corners
- Minimal clutter
- Friendly iconography
- Gentle animation
- Clean layouts with breathing room

### Overall Look
Think:
- Modern children’s wellness app
- Not babyish
- Not overly educational
- Not chaotic
- Polished enough for parents, delightful enough for children

---

## 3. Logo System

### Primary Logo Direction
A wordmark + symbol combination.

### Recommended Symbol Ideas
Choose one of these directions:

1. **Star + stepping path**
   - A star with a curved dotted path beneath it
   - Suggests guided progress and magical routine flow

2. **Moon + star + step**
   - A crescent moon wrapping around a star, with a subtle step/path element
   - Great for morning/evening duality

3. **Sunrise + star trail**
   - A star leaving a soft arc or stepping trail between moon and sun tones
   - Captures transition between bedtime and wake-up

### Logo Personality
The logo should be:
- Simple enough for app icon use
- Recognizable at small sizes
- Rounded and soft
- Slightly whimsical, but not cartoon-heavy

### Logo Variations
Create these versions:
- Primary horizontal logo
- Stacked logo
- Icon only
- Monochrome logo
- Favicon / app icon

### App Icon Direction
Best option:
- Rounded square background
- Single star or moon-star symbol
- Subtle path/step curve
- Minimal detail for clarity at small sizes

## 4. Color Palette

The palette should support both **morning energy** and **evening calm**.

### Primary Brand Colors

| Color Name      | Hex       | Usage |
|----------------|-----------|-------|
| Twilight Navy  | `#243B6B` | Headings, primary brand anchors, trust, contrast |
| Sky Blue       | `#6FAFEA` | Buttons, highlights, friendly UI moments |
| Stardust Gold  | `#F6C85F` | Stars, cheerful accents, reward moments |
| Moon Cream     | `#FFF6E8` | Warm backgrounds, cards, soft sections |
| Lavender Mist  | `#C9B8F5` | Bedtime flows, secondary accents, gentle magical feel |
| Mint Glow      | `#8ED9C4` | Success states, progress, positive reinforcement |

### Secondary Support Colors

| Color Name   | Hex       | Usage |
|--------------|-----------|-------|
| Soft Coral   | `#FF8E7A` | Friendly alerts, character moments, playful emphasis |
| Cloud Gray   | `#E9EEF5` | Borders, dividers, disabled states, subtle surfaces |
| Deep Night   | `#18253F` | Dark text, dark mode foundations, strong contrast |

### Suggested Palette Usage Ratio
- **40%** Moon Cream / light neutral backgrounds
- **25%** Twilight Navy
- **15%** Sky Blue
- **10%** Lavender Mist
- **5%** Stardust Gold
- **5%** Mint Glow / Coral accents

This keeps the brand polished rather than overly loud.

## 6. Typography Recommendations

You want typography that feels:
- Clean
- Rounded
- Modern
- Easy to read
- Child-friendly without looking childish

### Primary Font Recommendation: Nunito
Why it works:
- Rounded and friendly
- Highly legible
- Polished enough for UI
- Great for family/kids products

**Use for:**
- Headings
- Navigation
- Buttons
- Short UI labels

### Secondary Font Recommendation: Inter
Why it works:
- Excellent readability
- Modern and elegant
- Pairs beautifully with rounded display fonts
- Strong for app/product interfaces

**Use for:**
- Body text
- Settings
- Forms
- Instructions
- Parent-facing content

### Alternative Premium Feeling Pairings

#### Option A
- **Headings:** Nunito
- **Body/UI:** Inter

This is the strongest overall recommendation.

#### Option B
- **Headings:** Quicksand
- **Body/UI:** Inter

More playful, slightly softer, less premium than Nunito.

#### Option C
- **Headings:** Baloo 2
- **Body/UI:** Nunito Sans

Very playful, best only if you want stronger kid energy.

### Typography Scale Suggestion

#### Headings
- **H1:** Nunito Bold
- **H2:** Nunito Bold
- **H3:** Nunito SemiBold

#### Body
- **Body Large:** Inter Regular
- **Body Standard:** Inter Regular
- **Small Labels:** Inter Medium

#### Buttons
- Nunito SemiBold or Bold
- Slightly larger than standard UI text
- Sentence case preferred

## 7. Logo Typography Direction

For the **StarrySteps** wordmark, use:
- **Nunito Bold** or **Nunito ExtraBold**

Consider slight customizations:
- Rounded star shape in the “a” or dot accents
- Gentle curved baseline details
- A custom star replacing a dot if using title treatment

Keep it clean and avoid too many decorative substitutions.

## 8. UI Component Style

### Buttons
Buttons should feel tactile and happy.

#### Primary Buttons
- **Fill:** Sky Blue `#6FAFEA`
- **Text:** White `#FFFFFF`
- **Radius:** 16–24px
- **Optional shadow:** `0 6px 16px rgba(36, 59, 107, 0.15)`

#### Secondary Buttons
- **Fill:** Moon Cream `#FFF6E8`
- **Border/Text:** Twilight Navy `#243B6B`

#### Reward / Success Buttons
- **Fill:** Mint Glow `#8ED9C4`
- **Text:** Deep Night `#18253F`

#### Special Accent Buttons
- **Fill:** Stardust Gold `#F6C85F`
- Use sparingly for achievements, celebratory actions

### Cards
- Rounded corners: 20–28px
- Background: White or Moon Cream
- Soft shadows, never harsh
- Icons or illustrations at top
- Good spacing for touch interactions

### Inputs / Forms
- Rounded pill or soft rectangle style
- Border: Cloud Gray `#E9EEF5`
- Focus state: Sky Blue `#6FAFEA`
- Error state: Soft Coral `#FF8E7A`

- Rounded
- Simple
- Slightly thick stroke
- Recognizable for children
- Clean enough for parents

### Good icon subjects

- Pajamas
- Breakfast bowl
- Backpack
- Moon
- Sun
- Star
- Book
- Bath
- Shoes
- Toy cleanup

## 10. Illustration & Animation Direction

### Illustration Style
- Minimal, soft, and geometric
- Friendly faces if using characters
- Soft blush/cheek details optional
- Light sparkle details
- Calm visual storytelling

### Animation Style
Animations should be:
- Short
- Gentle
- Rewarding
- Not overstimulating

#### Good animation ideas
- Star twinkle
- Soft bounce on completed tasks
- Path dots lighting up
- Moon glow at bedtime
- Sun rise on morning routine completion

### Motion Principles
- Use easing
- Keep durations around 200–500ms for UI feedback
- Celebration animations can go slightly longer
- Avoid frantic movement

## 13. Accessibility Recommendations

Since this is a family product, accessibility matters a lot.

### Important Guidance
- Maintain strong contrast for text
- Don’t rely on color alone for task status
- Use large tap targets
- Support readable font sizes
- Pair icons with labels
- Offer calm visual mode for sensory-sensitive users

### Contrast Pairing Suggestions
Good text pairings:
- Twilight Navy on Moon Cream
- White on Twilight Navy
- Deep Night on Mint Glow
- Deep Night on Stardust Gold

Use caution with:
- Lavender Mist + white text
- Sky Blue + white text for small text unless darkened
